# Student Performance Analysis Dashboard

## Project Overview
This project analyzes student academic performance and attendance using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Key Insights
- Identified top-performing students.
- Analyzed pass and fail percentages.
- Compared performance across courses.
- Monitored attendance trends.

 ## Dashboard Features
- Total Students KPI
- Average Marks KPI
- Pass Percentage KPI
- Attendance KPI
- Interactive Slicers

## Skills Demonstrated
Data Cleaning, Data Analysis, Excel Functions, Pivot Tables, Dashboard Creation, Business Reporting.
