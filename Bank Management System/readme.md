## Sales Performance Dashboard
 Overview

This project is an Excel-based Sales Performance Dashboard created to analyze sales data and visualize key business metrics.

 Features

* Total Revenue Analysis
* Total Orders Tracking
* Top Product Identification
* Revenue by Product
* Revenue by Region
* Sales Person Performance Analysis
* Interactive Dashboard with Charts

 Tools Used

* Microsoft Excel / WPS Spreadsheet
* Pivot Tables
* Pivot Charts
* Excel Formulas

 Key Insights

* Top Product: Laptop
* Top Performing Region: North
* Top Sales Performer: Rahul
* Total Revenue: 1,572,300
* Total Orders: 50
